Pasos realizados:
1. Se creó el proyecto Django con 'django-admin startproject blog_project'.
2. Se creó la aplicación 'blog_app' para manejar autores y artículos.
3. Se configuró la base de datos Postgres en settings.py con credenciales.
4. Se definieron los modelos Autor y Articulo en models.py.
5. Se ejecutaron las migraciones con 'makemigrations' y 'migrate'.
6. Se insertaron registros y se realizaron consultas ORM desde el shell.
7. Se tomaron capturas de pantalla de las consultas para evidencias.
